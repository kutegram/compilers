SetupEmu_xxx.cmd provides easy-to-use emulator (winscw udeb) drive configurations.


SetupEmu_PhoneMemMmc.cmd            => Basic configuration (phone memory, memory card) 
SetupEmu_PhoneMemMassStorageMmc.cmd => Internal mass storage configuration (phone memory, internal mass storage, memory card)
SetupEmu_PhoneMemMmc1Mmc2.cmd       => Two removable memory card configuration (phone memory, memory card 1, memory card2)


USAGE:

1) Make sure that Code Warrior and emulator are closed (i.e. files under epoc32\release\winscw are not in use).
2) Run SetupEmu_xxx.cmd
3) Start emulator
