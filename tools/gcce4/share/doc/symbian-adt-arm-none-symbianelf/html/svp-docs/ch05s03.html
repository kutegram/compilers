5.3. Device Object Methods

Device objects provide the following methods to assist with implementing device emulation:

set_irq_level(irq, level)

Rise or lower a device IRQ output. irq is a zero based index specifying which IRQ to modify. level is zero to lower the IRQ or one to raise it.

create_interrupts(callback, count)

Create input IRQ lines (for interrupt controller type devices). When the IRQ line state changes the function callback(irq, level) will be called.

dma_readb(address)

Read a byte from system memory.

dma_writeb(address, value)

Write a byte to system memory.
