5.4. Snapshots

In order to support snapshots, devices must be able to save and restore internal state. The is achieved by overriding the save and load methods. The snapshot state can be accessed via the object passed to these methods. This state object provides the following methods:

get_u32()

Read an unsigned 32-bit integer.

get_u64()

Read an unsigned 64-bit integer.

get_s64()

Read a signed 64-bit integer.

put_u32()

Write an unsigned 32-bit integer.

put_u64()

Write an unsigned 64-bit integer.

put_s64()

Write a signed 64-bit integer.
